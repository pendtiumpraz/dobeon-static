| Halaman | Route | Konten |
|---------|-------|--------|

## SEO & Performance

- Metadata komprehensif per halaman
- Open Graph & Twitter Card
- Structured data JSON-LD (organisasi, layanan, tim)
- Sitemap XML & robots.txt
- PWA manifest
- Code splitting & optimasi gambar
- Kompresi Gzip/Brotli
